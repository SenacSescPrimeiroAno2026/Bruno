Flexbox Froggy🐸

Descrição:

é um jogo educativo gratuito com 24 níveis, projetado para ensinar propriedades CSS Flexbox, como alinhamento e direção, através de uma interface interativa. Os jogadores posicionam sapos em vitórias-régias usando código, tornando-o uma ferramenta prática e recomendada para iniciantes no desenvolvimento web.
