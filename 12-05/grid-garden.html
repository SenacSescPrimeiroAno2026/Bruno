Grid garden🪴

Descrição:

O Grid Garden é um jogo educativo gratuito e interativo, criado pela Codepip, projetado para ensinar os fundamentos do CSS Grid Layout de forma divertida. No jogo, você escreve códigos CSS para regar e cultivar um jardim de cenouras, utilizando propriedades como grid-column-start e grid-template-columns para posicionar elementos em uma grade.
